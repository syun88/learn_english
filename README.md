# English Grammar & Verb Master JP / 繁中

英語をゼロから学び直すための、日本語・繁體中文バイリンガル文法サイトです。

単語帳ではなく、まず英語の文法構造と動詞変化を固めることを目的にしています。

## Site

GitHub Pages URL:

```text
https://syun88.github.io/learn_english/
```

表示されない場合は、GitHub の `Settings` → `Pages` で、Source を `Deploy from a branch`、Branch を `main`、Folder を `/root` に設定してください。

## Contents

- 2026 年内ロードマップ
- 文型：SV / SVC / SVO / SVOO / SVOC
- 品詞の全体像
- be / do / have の役割
- 動詞変化：原形・三単現・過去形・過去分詞・ing形
- 肯定文・否定文・疑問文・WH疑問文
- 12時制の表とタイムライン
- 助動詞
- 不定詞・動名詞・分詞
- 名詞節・関係詞・副詞節
- 受動態・比較・仮定法
- 日本語・中国語母語話者が間違えやすい点
- 例文検索

## Policy

- Vocabulary first ではなく grammar first
- 練習問題より例文と使い方重視
- 日本語と繁體中文を同時表示
- 外部ライブラリなしで GitHub Pages に対応
